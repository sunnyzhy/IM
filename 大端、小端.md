# 大端、小端

## 大端模式

大端模式（Big-Endian）又称为网络字节序，指的是数据的高位字节存储在低地址处，而数据的低位字节存储在高地址处。这与我们平常的阅读顺序相同，其实大端模式才是我们直观上认为的模式。

大端模式下，一个4字节的整数 ```0x12 34 56 78``` 在内存中的表示形式：

```
低地址 ------------------> 高地址
0x12  |  0x34  |  0x56  |  0x78
```

```
高地址
        ---------------
        0x78          -- 低位
        0x56
        0x34
        0x12          -- 高位
        ---------------
低地址
```

## 小端模式

小端模式（Little-Endian）指的是数据的低位字节存储在低地址处，而数据的高位字节存储在高地址处。这与我们平常的阅读顺序相反。

```
低地址 ------------------> 高地址
0x78  |  0x56  |  0x34  |  0x12
```

```
高地址
        ---------------
        0x12          -- 高位
        0x34
        0x56
        0x78          -- 低位
        --------------
低地址
```
